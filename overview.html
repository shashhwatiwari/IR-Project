Project Overview

Our team project involves using a breached phone number dataset to locate where the phone numbers have been leaked to. To achieve this, we will be applying a range of information retrieval techniques, including data mining, natural language processing, and machine learning. These techniques will help us extract and classify the relevant data based on the phone number's location. Throughout the project, we may encounter technical terms such as indexing, querying, ranking, and relevance to aid in our search process.

Website Development

To present our findings, we plan to create a website using HTML, CSS, and JavaScript. HTML will help us structure the website's content, CSS will provide an appealing design, and JavaScript will add interactive features and functionality to the site. Additionally, we will use a server-side programming language such as PHP to manage the website's backend functionality.

Information Retrieval Techniques

  • Data mining
  • Natural language processing
  • Machine learning
